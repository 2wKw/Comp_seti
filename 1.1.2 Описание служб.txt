
Интернет можно описать рассматривая его, как инфраструктуру, которая предоставляет 
                службы для приложений.

Распределенные приложения - это приложения, которые вовликают в себя множество
                конечных систем, которые в своё время обмениваются данными друг с другом.

Интерфейс программирования приложений ( Application Programming Interface, API) - API
                интернета яфвляется набором правил, которые должны выполняться, чтобы 
                интернет мог доставлять данные в целевую программу
