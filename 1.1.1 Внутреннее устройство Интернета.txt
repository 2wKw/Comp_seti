
Интернет - это компьютерная сеть, которая связывает между собой сотни миллионов 
                  вычислительныйх устройств по всему миру.

Хосты(конечные системы) - устройства, которые подключаются к интернету.

Линии связи -  физическая среда, по которой передаются информационные сигналы,
                         формируемые специальными техническими средствами, относящимися
                         к линейному оборудованию

Скорость передачи - объём данных передаваемый за единицу времени.

Пакеты - это определенным образом оформленных блок данных, передаваемый по сети.

Маршрутизатор (router) - это устройство, которое получает интернет от провайдера и 
                                         передает его на устройства подключённые к внутренней сети.

Коммутатор (switch) -  устройство управления, которое включает или выключает поток 
                                     электроэнергии в цепи. Он также может использоваться 
                                     для маршрутизации информационных шаблонов при потоковой 
                                     передаче электронных данных, передаваемых по сетям.

Интернет-провайдер (Internet Service Providers, ISP) - то компания, которая, обеспечивает 
                                   доступ клиента к Сети интернет. Современные провайдеры 
                                   предоставляют и другие услуги: телевидение, мобильную связь, 
                                   online-сервисы для просмотра фильмов и хостинг.

Протокол - это, образно говоря, «язык», используемый компьютерами для обмена данными 
                   при работе в сети. Чтобы различные компьютеры сети могли взаимодействовать, 
                   они должны «разговаривать» на одном «языке», 
                   то есть использовать один и тот же протокол.

Протокол управления передачей (Transmission Control Protocol, TCP) - один из основных 
                   протоколов передачи данных интернета. Предназначен для управления
                   передачей данных интернета. Пакеты в TCP называются сегментами.

Протокол Интернета (Internet Protocol, IP) - Маршрутизируемый протокол сетевого уровня 
                  стека TCP/IP. Именно IP стал тем протоколом, который объединил отдельные 
                  компьютерные сети во всемирную сеть Интернет. Неотъемлемой частью 
                  протокола является адресация сети.

Интернет стандарты - разрабатываются инженерным советом Интернета (IETF, Internet
                  Engineering Task Force)
